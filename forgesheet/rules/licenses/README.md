# Content Licensing

ForgeSheet ships only with legally safe sample content and short-form open-compatible rules metadata.

- `rules/srd`: open-license-compatible starter content and concise references.
- `rules/custom`: reserved for project-authored original content.
- `rules/homebrew`: reserved for user-authored imports.
- Proprietary 5e text is not bundled.
- Long-form copyrighted descriptions must be provided by the user or through a licensed ingestion pipeline.
