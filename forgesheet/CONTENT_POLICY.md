# Content Policy

ForgeSheet ships only with legally safe bundled content.

## Included by Default

- open-compatible structured metadata
- concise starter summaries
- project-authored UI text and visuals

## Not Included by Default

- proprietary 5e long-form text
- copyrighted class, spell, or feature descriptions beyond safe summary form
- licensed expansion text without authorization

## Planned Ingestion Paths

- `rules/homebrew` for user-authored packs
- `rules/custom` for project-authored original content
- future licensed pack manifests through `ContentPack`

Every content source should carry licensing metadata and provenance before import.
